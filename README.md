# BlogCuisine

BlogCuisine (La cuisine familiale) is a website for sharing our family recipes.

## Technologies

Frontend :

- HTML
- CSS
- JavaScript
- Bootstrap

Backend :

- PHP
- MySQL

### FEATURES

- Responsive design
- User registration and login
- Add, edit, and delete recipes
- Search for recipes
- Sort recipes
